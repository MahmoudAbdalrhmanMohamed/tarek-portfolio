# Tarek Mostafa Irrigation Portfolio

Static bilingual portfolio website for Tarek Mostafa, Agricultural Engineer and Site/Technical Office Irrigation Engineer.

The portfolio presents irrigation design work, project package downloads, generated project previews, SEO metadata, and contact links.

## Contents

- `index.html` - Main bilingual portfolio page with English/Arabic switching.
- `profile-photo.jpg` - Hero portrait image.
- `assets/garden-preview.png` - Preview generated from the Garden project layout PDF.
- `assets/central-preview.png` - Preview generated from the Central Park project layout PDF.
- `Tarek_Mostafa_Agricultural Engineer_CV.pdf` - Downloadable CV.
- `Garden.rar` - Garden project package.
- `Central park - Tarek Mostafa.rar` - Central Park project package.
- `favicon.svg` and `seo-preview.svg` - Browser icon and social preview artwork.
- `robots.txt`, `sitemap.xml`, `site.webmanifest` - SEO/PWA metadata files.

## Features

- Full-screen hero section with fixed glass navigation.
- English and Arabic language switch with RTL support.
- Smooth anchor scrolling and scroll progress indicator.
- Custom themed scrollbar.
- Responsive layout for desktop, tablet, and mobile.
- Project showcase cards with real PDF-generated previews.
- Package-level downloads for CV, Garden, and Central Park.
- SEO tags, Open Graph/Twitter preview metadata, and JSON-LD structured data.

## Preview Locally

Open `index.html` directly in a browser.

No build step or package installation is required.

## Deployment Notes

Before publishing, replace the placeholder domain:

- `https://tarek-mostafa-portfolio.example/`

Update it in:

- `index.html`
- `robots.txt`
- `sitemap.xml`
